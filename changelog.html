Mudanças e melhorias


Versões 0.4 - Melhoramentos de performance e engine de partida

Versão 0.4.3

Atualizações na engine de partida e criação do histórico de partidas anteriores, acessado pela tela inicial do usuário.

Versão 0.4.2

Melhorias gerais de performance.

  • Carregamento médio de páginas entre 0.5 e 1s mais rápidos;
  • Correção de bugs na utilização do cache interno;
  • Adicionadas mais informações sobre os times na página de partida;
  • Melhorada segurança do login / sessão do usuário.

Versão 0.4.1

Algumas melhorias de usabilidade e na partida.

  • Corrigido bug que obrigava o jogador a resetar o time toda vez que fizesse uma venda de um jogador titular;
  • Adicionada a opção início no menu superior;
  • Melhorada a interface das tabelas em telas grandes;
  • A partida fica salva depois que é rodada uma primeira vez, importante para criação de rankings no futuro.

Versão 0.4.0

Salvar time agora salva os dados do time no banco de dados e sempre que um time é organizado, algumas das novidades:

  • Jogadores títulares, formação, modo de jogo e estilo de ataque ficam salvos;
  • Melhorias de acessibilidade;
  • Quando um jogador é vendido, se estiver no time titular, ele é retirado na página de "Meu time".

Versões 0.3 - Compra e venda

Versão 0.3.3

Melhorias diversas de usabilidade. Os destaque são:

  • Confirmações para venda de jogadores e resetar o time;
  • Ícones no menu inferior;
  • Melhorado o calculo de posse de bola dos times;
  • Novo estilo de botão.

Versão 0.3.2

Correções de erros e melhorias de layout.

Versão 0.3.1

Criado o mercado da bola, dentre outras novidades:

  • Correção da visualização de tabelas em dispositivos móveis iOS;
  • Compra de jogadores no mercado da bola;
  • Opções de navegação pelo menu superior;
  • Adiciona um menu de navegação inferior para dispositivos móveis;
  • Adicionados novos jogadores de times diversos para tornar o jogo mais equilibrado.

Versão 0.3.0

Adicionada a possibildiade de vender jogadores.

  • Criadas regras de número mínimo de jogadores no elenco e jogadores por posição;
  • A venda do jogador automaticamente adiciona o valor da venda no caixa do time.

Versões 0.2 - O painel de instrumentos

Versão 0.2.1

Correções de erros e melhorias, os destaques são:

  • Botão de resetar jogadores voltou a funcionar;
  • Adicionado tempo nas jogadas.

Versão 0.2.0

Criado um Dashboard como tela principal. Algumas das novidades:

  • Lista de todos os jogadores do elenco;
  • Dados financeiros do time;
  • Dados acessados de forma muito mais rápida.

Versões 0.1 - Credenciais

Versão 0.1.1

Pequenos ajustes diversos:

  • Melhoradas algumas das mensagens de alerta;
  • Adicionadas mais comentários durante o jogo;
  • Ajustado erro que permitia adicionar um jogador mais de uma vez no mesmo time;
  • Agora a formação 3-5-2 está funcionando.

Versão 0.1.0

Adicionado sistema de login e credenciais, com cadastro de usuários pelo Google Sign-in.


Versões 0.0 - O começo

Versão 0.0.1

As principais novidades foram:

  • Escolhido o nome 1,2,3 GOL!;
  • Adicionados 9 jogadores do elenco do Palmeiras no banco de dados;
  • Novo tipo de jogada possível, onde o time nem consegue dominar a bola e começar um ataque;
  • Automatizadas as notas dos jogadores a partir de um cálculo baseado em suas características;
  • Pequenos ajustes de layout;
  • Criada a página de mudanças e melhorias.

Versão 0.0.0

Iniciado o desenvolvimento.

Criada a primeira versão da Engine de partida, além da área para montar o time para a partida. De início todos os jogos são contra um único time adversário pré-definido.

Na primeira versão estão disponíveis 18 jogadores do elenco do Corinthians.
